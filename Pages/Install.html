We provide two different installation methods: Standard Installations and Jar file. If your platform has a Standard Installation, then please use that. If your platform does not (or the Standard Installation breaks), then please refer to the Jar instructions.

Jar Instructions

  • Please install the latest Jar file here.
  • Jar files can be run on any OS with a Java Virtual Machine (JVM). As such, please install a JVM here.
    You should use Java version 23.0.2 for best results (you can check your version by running java --version in a terminal/command-prompt).
  • Navigate to where your Jar file is stored. Open a terminal at the folder where the Jar file is stored, and run the following command: 
                        
                        java -cp nameOfJarFile.jar uk.ac.warwick.dcs.junctionmodeller.App
    Note you will have to replace "nameOfJarFile.jar" with the name of your Jar file
